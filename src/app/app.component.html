AG-Grid Filter Option Display Bug

Bug Reproduction Steps:

  1. Select a specific make from the "Make" column (i.e. Ford).
  2. Click the "Remove {make} From Data" button on the bottom of the form.
  3. Click in the filter to see that the option is no longer selected or available, but table data is empty.
  4. At this point the user would have to re-select a filter in order to see data again.
  5. Click the "Show All Data" button.
  6. Click in the filter to see that all options are available again, but "Ford" is not selected.
  7. At this point the user would also have to re-select a filter in order to see data again.
